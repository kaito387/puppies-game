# Puppies-Game 需求分析报告

**项目名称**: Puppies-Game（狗国建设者）
**项目类型**: Web 端增量式游戏

---

## 1. 项目概述

### 1.1 项目描述

Puppies-Game（狗国建设者）是一款 Web 端的增量式游戏（Idle Game），玩家通过点击、建造建筑、研究技术等方式管理一个不断增长的狗狗殖民地。游戏采用自动化的 tick 机制，使玩家可以在长时间内保持游戏运行，逐步解锁新内容并提高资源生产效率。

### 1.2 项目目标

- **主要目标**: 构建一款功能完整、高质量的增量式游戏，提供沉浸式的游戏体验
- **学习目标**: 掌握现代Web开发技术栈和软件工程实践
- **交付目标**: 按时交付满足课程要求的所有 9 项交付物，包括代码、文档、报告和演示

---

## 2. 需求背景与可行性分析

### 2.1 市场需求

增量式游戏（Idle Games）是一类流行的网络游戏，具有：

- 低操作难度，适合休闲玩家
- 长期的参与度，玩家可持续多月或多年
- 简洁的美术需求，适合小团队开发
- 广泛的用户基础（Kittens Game 等成功案例）

### 2.2 技术可行性

- **技术栈成熟**: React 18 + TypeScript + Vite 均为业界标准技术，生态完善
- **团队技能匹配**: 学生具备基本的 Web 开发能力
- **开发周期合理**: MVP 可在 5-7 天内完成，完整版本在 14 周内完成
- **部署简便**: 可直接部署到 GitHub Pages，无需专业服务器
