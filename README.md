# Puppies-Game (狗国建设者)

🐕 一款浏览器 Web 增量游戏 (Idle Game)，参考 [Kittens Game](https://kittensgame.com/)，用现代 Web 技术实现。

玩家通过建设、研究、管理资源来逐步扩展自己的狗狗帝国。

## 快速开始

### 安装依赖

```bash
npm install
```

### 构建生产版本

```bash
npm run build
npm run preview  # 预览构建结果
```
